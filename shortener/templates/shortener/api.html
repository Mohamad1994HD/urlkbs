An Application Programming Interface (API) defines interfaces to a programming library or framework for accessing functionality provided by the library or framework. The KBS REST API provides methods for accessing the shortening feature in our product. Your program talks to KBS using HTTP, the same protocols that your web browser uses to interact with web pages, and follows the principles of Representational State Transfer (REST).

Endpoint "api.urlkbs.life/" for post request

Usage (SIMPLE)

post request to "api.urlkbs.life" with "url" as parameter

cURL

    
        curl --data "url=www.abc.com" http://api.urlkbs.life/

Python


import requests
r = requests.post('http://api.urlkbs.life/', data={'url':'https://django-hosts.readthedocs.io/en/latest/'})

if r.json()['accepted']:
    print r.json()['data']['shortcode']

SUCCESS

  'accepted': 1,
  'data': {

            'url':  'https://django-hosts.readthedocs.io/en/latest/'
            }
}
