An Application Programming Interface (API) defines interfaces to a programming library or framework for accessing functionality provided by the library or framework. The KBS REST API provides methods for accessing every feature in our product. Your program talks to KBS using HTTP or HTTPS, the same protocols that your web browser uses to interact with web pages, and follows the principles of Representational State Transfer (REST).

Usage

post request to "api.kbs.com" with "url" as parameter

cURL

    
        curl --data "url=www.abc.com" http://api.kbs.com

Python


import requests
r = requests.post('http://api.kbs.com/', data={'url':'https://django-hosts.readthedocs.io/en/latest/'})

if r.json()['accepted']:
    print r.json()['data']['shortcode']

Response

SUCCESS

JSON

    
{
  'accepted': 1,
  'data': {
            'shortcode': 'http://www.kbs.com/xHTOog/',
            'url':  'https://django-hosts.readthedocs.io/en/latest/'
            }
}

JSON

    
    {'accepted': 0, 'msg': 'Invalid URL'}
